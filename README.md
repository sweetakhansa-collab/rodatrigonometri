# Roda Trigonometri Interaktif

Project ini merupakan media pembelajaran matematika yang dibuat dalam bentuk website interaktif.

Pengguna dapat menentukan:
- Sudut
- Fungsi trigonometri (sin, cos, tan, sec, cosec, cot)

Cara menggunakan:
1. Geser panah sudut ke sudut yang diinginkan.
2. Geser panah fungsi ke fungsi trigonometri yang ingin dicari.
3. Hasil akan muncul di bagian tengah.

Tujuan project:
Membantu siswa memahami hubungan antara sudut dan nilai fungsi trigonometri secara visual dan interaktif.

Dibuat menggunakan:
- HTML
- CSS
- JavaScript
